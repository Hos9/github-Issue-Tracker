- 1️⃣ What is the difference between var, let, and const?
- Answer: "var" is old method to declare parameters it is automatically Hoisted. "let" & "const" also declare variables also the main difference is "let" can be changeable but "const" is not changeable.

- 2️⃣ What is the spread operator (...)?
- Answer: Iterables (strings, arrays) or object properties can be expanded into individual elements or properties using the spread operator.

- 3️⃣ What is the difference between map(), filter(), and forEach()?
- Answer: map(): transforms each item and returns a new array (same length).
  filter(): keeps only items matching a condition and returns a new (smaller) array.
  forEach(): runs a function for each item and returns nothing (undefined), used only for side‑effects.

- 4️⃣ What is an arrow function?
- Answer: Lexical this (doesn't bind its own this; it gets it from the surrounding scope). Good for callbacks. There is no arguments object (use rest ...args if you need one). Not able to be used as constructors (new). Short return for single expressions.

- 5️⃣ What are template literals?
- Answer: Strings with backticks `` that support:
  Interpolation: ${expression}, Multiline strings & Tagged templates (advanced)
